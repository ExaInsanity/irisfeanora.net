Helium Toolchain

The Helium Toolchain consists of three main projects, all predominantly using C#: Helium, Oxygen and Xenon.

To protect ourselves from being besieged, they are slowly being revealed as we see fit. So far, Xenon is public.

Xenon

Xenon is a Minecraft launcher, written in C#, and is intended to be as powerful or more powerful than MultiMC, as easy-to-use as the Mojang Launcher and support running servers as well as third-party software which doesn't use Java or the notchian command-line argument syntax.
